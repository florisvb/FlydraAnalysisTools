===========
Flydra Analysis Tools
===========

Flydra Analysis Tools provides tools for analyzing and plotting large datasets from multiple h5 files in a picklable class and dictionary structure. See examples for a comprehensive example that covers most of the code.

To use the plotting functions as they are written you will need to install the following packages (allows saving adobe illustrator friendly pdfs)
texlive ghostscript dvipng texlive-latex-extra
